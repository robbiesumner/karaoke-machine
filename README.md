# karaoke-machine

A plug-and-play Linux distribution for hosting karaoke parties.
Think *Batocera, but for [UltraStar Deluxe][usdx]*.

Boot the ISO on a PC, plug in SingStar mics and a TV, drop songs into a
network share, and start singing.

> **Status:** pre-release, in active development. v1.0 is the first stable
> ISO — see [ROADMAP.md](ROADMAP.md) for current progress.

## Quickstart
